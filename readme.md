# Стартовый шаблон

Данный шаблон включает:
1. React + TS
2. eslint + prettier
3. react-router
4. emotion
5. react-toastify
6. axios
7. formik
8. yup
9. faker-js
10. framer-motion
11. vitest
12. jest

dev версия запускается командой npm run dev / yarn dev
Для тестов используется команда npm run test / yarn test
